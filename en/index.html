Silly

A lightweight, minimalist high-performance Lua server framework.

Getting StartedAPI Reference

Ready to Use

  • Implement high-concurrency Echo server in 10 lines of code

Rich Features

Built on top of Lua, tons of features added for you to easily build high-concurrency servers.

silly.core

Provides coroutine scheduling operations

silly.net

Provides tcp/udp/tls network operations

silly.store

Provides redis/mysql/etcd storage system operations

silly.sync

Provides locks, queues, inter-coroutine communication and other synchronization operations

silly.security

Provides JWT and other security features

silly.metrics

Provides prometheus monitoring support

silly.crypto

Provides common cryptographic algorithms

silly.encoding

Provides json/base64 encoding and decoding support

silly.console

Provides console command line support

silly.debugger

Provides online debugger support

silly.logger

Provides logging support

silly.patch

Provides hot reload support

silly.adt

Provides efficient data structures (buffer, queue, etc.)

zproto

Provides protocol encoding and decoding support

pb

Provides protobuf encoding and decoding support
