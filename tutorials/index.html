教程

欢迎来到 Silly 框架教程！本部分提供循序渐进的学习路径，帮助新手快速上手。

推荐学习路径

  1. 快速开始 - 10分钟快速上手 Silly 框架
  2. Echo 服务器 - 学习网络编程和协程
  3. HTTP 服务器 - 构建 RESTful API
  4. 数据库应用 - MySQL + Redis 完整应用
  5. WebSocket 聊天室 - 实时通信应用

教程特点

  • 学习导向: 通过实践学习核心概念
  • 循序渐进: 从简单到复杂
  • 可运行代码: 所有示例都经过验证，可以直接运行

  • 操作指南 - 解决具体问题
